This folder contains term papers/presentations submitted for different courses:

Bootstrap Summary: review of bootstrapping techniques, implementation of an outlier-detection technique using bootstrapping method

Graph Reachability: concept of graph reachability query, summarize different approaches that have been proposed to answer reachability 
		and compare those approaches on the basis of query processing time, storage required and their ease of implementation

Non-Fungible Tokens: concept of Non-Fungible Tokens (NFTs), their scope in modern economy, role of encryption in implementing NFTs

Rio Grande Cubic Spline Interpolation: using Numerical Analysis to fit continuous curve (that passes vertical line test) and
		interpolating its sampled data points to generate a function that approximates the Rio Grande curve.
